# Calculator Project

A beginner-friendly calculator project made with HTML, CSS, and JavaScript.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Percent/remainder operation
- Clear button
- Delete button
- Decimal support
- Keyboard input support
- Responsive calculator layout

## Files

- `index.html` - Calculator interface and project content
- `style.css` - Calculator design, display, button grid, and responsive layout
- `script.js` - Calculator logic, event listeners, and keyboard support

## How to Run

Open `index.html` in your browser.

If you use VS Code, you can also run it with the Live Server extension.

## Keyboard Shortcuts

- Numbers: `0` to `9`
- Operators: `+`, `-`, `*`, `/`, `%`
- Calculate: `Enter` or `=`
- Delete: `Backspace`
- Clear: `Escape`
